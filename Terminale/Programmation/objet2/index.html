    Programmation Orientée Objet (P.O.O.)#

    Objet et attribut#

    Un objet contient un ensemble d'attributs qui ont chacun une valeur, un peu comme dans un dictionnaire. On accède aux attributs d'un objet par la notation pointée : obj.attr est la valeur de l'attribut attr de l'objet obj.

    Par exemple, un point de l'espace à deux dimensions peut-être représenté par un objet avec deux attributs \(x\) et \(y\). Si pest un tel objet, on peut accéder à ses attributs par p.xet p.y.

    Pour créer un objet, on a besoin de définir une classe. La définition la plus simple est la suivante :

    1
2
    class Point
    pass

    Une fois la classe définie, on peut créer des instances de la classe en utilisant le nom de la classe de la manière suivante :

    1
2
3
    p1 = Point() # nouvel objet p1
p1.x = 10 #affectation de l'attribut x de p1
p1.y = 15 #affectation de l'attribut y de p1

    On dit que p1 est une instance de la classe Point ou que p1 est un objet Point.

    Les objets sont manipulés dans un programme Python par comme les autres valeurs du langages : on peut les passer en paramètres, accéder et et affecter leurs attributs et les retourner comme valeurs d'une fonction.

    Par exemple, la fonction suivante prend en paramètres deux instances p1 et p2 de la classe Point, crée une nouvelle instance m de la classe Point, affecte ses attributs et retourne cette nouvelle instance.

    1
2
3
4

    def milieu(p1, p2):
    """
    retourne un nouveau point, milieu du segment (p1,p2) 
    """
    m = Point()
    m.x = (p1.x + p2.x) / 2.0
    m.y = (p1.y + p2.y) / 2.0
    return m

    Attention :

    Comme les tableaux et les dictionnaires, les objets sont muables puisqu'on peut changer les valeurs de leurs attributs. Cela peut donc causer des surprises lorsque deux variables référencent le même objet (aliassage). Exemple :

def zero(p):
    p.x = 0
    p.y = 0

zero(p1) # p1 et p2 ont maintenant 
# pour attributs x = 0 et y = 0

    Classes et méthodes#

    Définition des méthodes#

    Le but d'une classe est de pouvoir créer et manipuler des objets qui ont des caractéristiques communes : les instances de la classe Point, par exemple, représentent des points du planet s'attend à pouvoir faire des opérations telles que translater un point ou calculer sa distance à un autre point.

    La classe permet de regrouper ces opérations en définissant des méthodes, c'est-à-dire des fonctions propres à cette classe, définies à l'intérieur de celle-ci.

    Par exemple, la classe Point pourrait avoir trois méthodes :

    • init(p,x,y) pour initialiser le point p aux coordonnées \((x, y)\) ;
    • translater(p, v)pour translater un point p d'un vecteur v ;
    • distance(p,q) pour calculer la distanc eentre les points pet q.

    Ces fonctions ont toutes la particularité que le premier paramètre est un Point. Conventionnellement, on appelle ce paramètre self lorsque l'on définit une méthode.

    Ainsi, la méthode init est définit de la manière suivant :

    1
2
3
4
    class Point:
    def init(self, coord_x, coord_y):
        self.x = coord_x
        self.y = coord_y

    Pour appeler cette méthode, on utilise la notation pointée, comme pour les attributs :

    1
2
    p = Point()
p.init(10, 15)

    IMPORTANT :

    • init n'est pas un attribut de p mais une méthode définie dans la classe Point. Le langage reconnaît que p est un objet et que init n'est pas un de ses attributs, il regarde alors si c'est une méthode définie dans sa classe.
    • Bien que l'on ait déclaré la méthode init comme prenant trois paramètres (self, x, y), on l'appelle avec deux paramètres (10et 15). Le paramètre selfcorrespond à l'objet p de la notation pointée p.init(...). En d'autres termes, p.init(x,y) appelle la méthode init(p, x, y) de la classe de p, c'est-à-dire de la classe Point

    Il résulte des deux remarques précédentes que :

    • toutes les instances d'une classe ont accès à ses méthodes : celles-ci définissent donc l'interface de la classe ;
    • deux classes peuvent avoir une méthode de même nom. Par exemple, la classe Point et la classe Segmentpeuvent toutes les deux avoir une méthode translater.

    def init(self, coord_x, coord_y):
        self.x = coord_x
        self.y = coord_y

    def translater(self, v):
        self.x += v.x
        self.y += v.y

    def distance(self, p2):
        return math.sqrt((p2.x - self.x)**2 + (p2.y -self.y)**2)

    Garantir l'initialisation avec un constructeur#

    Il est important de créer des objets dont l'état initial est connu. Dans les exemples précédents, la création d'une instance en faisant p = Point() crée un objet qui n'a aucun attribut et, à ce stade, on ne peut appeler de méthode sans créer des erreurs.

    1
2
3
4
5
6
7
    >>> p1 = Point()
>>> p2 = Point()
>>> p1.distance(p2)
Traceback (most recent call last):
    File "<stdin>", line 1, in <module> 
    File "<stdin>", line 12, in distance 
AttributError: 'Point' object  has no attribute 'x'

    Si l'on définit dans la classe une méthode au nom spécial __init__ , cette méthode appelée constructeur sera automatiquement appelée lorsque l'on crée un objet :

    1

    class Point:
    def __init__(self):
        self.x = 0
        self.y = 0
p = Point() # p a deux attributs x = 0 et y = 0

    Encapsulation#

    Le problème des accès aux données#

    On peut accéder aux attributs d’un objet p de la classe Point avec la notation p.x où a désigne le nom de l’attribut x visé.

    Pour rendre privé les attributs d’une classe, il faut les faire précéder de 2 tirets. Exemple, dans le constructeur de la classe Point, self.__x = 4.

    Remarque : on peut rendre privé aussi les méthodes en les faisant précéder de deux tirets.

    Les accesseurs ou "getters"#

    On ne va généralement pas utiliser la manière de procéder précédente nom_objet.nom_attribut permettant d’accéder aux valeurs des attributs car on ne veut pas forcement que l’utilisateur ait accès à la représentation interne des classes. Pour utiliser ou modifier les attributs, on utilisera de préférence des méthodes dédiées dont le rôle est de faire l’interface entre l’utilisateur de l’objet et la représentation interne de l’objet (ses attributs). Les attributs sont alors en quelque sorte encapsulés dans l’objet, c’est à dire non accessibles directement par le programmeur qui a instancié un objet de cette classe.

    Pour obtenir la valeur d’un attribut nous utiliserons la méthode des accesseurs (ou "getters") dont le nom est généralement : get_Nom_attribut(). Par exemple ici :

    class Point:
    def __init__(self, coord_x, coord_y):
        self.x = coord_x
        self.y = coord_y

        return self.x

        return self.y

p = Point(5,18) # p a deux attributs x = 5 et y = 18

    les mutateurs ou "setters"#

    En cas de besoin, on peut modifier les valeurs attribuées aux attributs. Pour cela, on passe par des méthodes particulières appelées mutateurs (ou "setters") qui vont modifier la valeur d’une propriété d’un objet. Le nom d’un mutateur est généralement : set_nom_attribut().

    1
2
    def set_x(self, x2):
    self.x = x2

    Exercice : Complexe

    En mathématiques, dans un plan rapporté à un repère orthornormé \((O,\vec{u},\vec{v})\) , tout point \(M\) de coordonnées \((x ; y)\) peut être représenté par ce que l'on appelle un nombre complexe qui peut s'écrire sous la forme :

    \[ z = x + i y \]

    On dit que \(x\) est la partie réelle de z et y est sa partie imaginaire.

    En posant \(z=x+iy\) et \(z'=x'+iy'\), on définit les opérations suivantes :

    • \(z+z'= (x+x')+i(y+y')\)
    • \(z-z'= (x-x')+i(y-y')\)
    • \(z\times z'= (xx'-yy')+i(xy'+x'y)\)

    De plus, on dit que \(z=z'\) si \(x=x'\) et \(y=y'\)

    Écrire en python, une classe complexe :

    • qui définit un nombre complexe (le constructeur devra initialiser un tuple de deux nombres : la partie réelle et la partie imaginaire) ;

    • ayant une méthode permettant d'afficher le nombre complexe sous forme d'un tuple de deux éléments ;

    • permettant d'ajouter, soustraire, multiplier et comparer (en termes d'égalité) deux nombres complexes ;

    • permettant de donner la distance de l'origine du repère au point représenté par le point complexe (on appelle cette distance le module, qui est égale à \(\sqrt{x^2+y^2}\) );

    Tester cette classe avec les nombres :

    • \(z=-3+5i\)
    • \(z'=7+i\)

    Exercice : classe Angle

    Définir une classe Angle pour représenter un angle degrés. Cette classe contient un unique attribut, angle qui est un entier. On demande quoi qu’il arrive, que l’égalité 0 ≤ angle < 360 reste vérifiée.

    5. Tester les méthodes ajouter, cosinus et sinus.
