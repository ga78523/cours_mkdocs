    Quel type de verrerie est le plus précis ?#

    Objectif du TP#

    Lors d’une intervention chirurgicale, un patient doit recevoir une perfusion de glucose afin d’avoir un apport énergétique suffisant pour l’opération. Le volume de solution à préparer est de 100 mL. Nous disposons de 3 éléments de verrerie portant l’indication 100 mL :

    • 1 Bécher ;
    • 1 Éprouvette graduée ;
    • 1 Fiole jaugée.

    Problématique#

    Notre objectif est de déterminer quel élément de verrerie choisir afin de préparer la solution avec un maximum de précision.

    Partie Pratique#

    Vous répondrez aux questions sur un document texte.

    Pour répondre à la problématique, nous allons effectuer 3 prélèvements de 100 mL d’eau avec chacun des 3 éléments de verrerie. Afin de vérifier si à chaque fois on a bien prélevé le même volume d’eau, nous allons le peser avec une balance de précision.

    Travail à réaliser en binôme#

    • Reproduire sur votre compte-rendu, le tableau ci-dessous.

    Pour chacun des instruments (bêcher, éprouvette graduée, fiole jaugée) :

    • Poser l’instrument bien sec sur une balance de précision et effectuer la tare ;
    • Le remplir d’eau jusqu’à la graduation 100 mL. Le niveau pourra être affiné très précisément à l’aide d’une pipette pasteur ;
    • Noter sur votre compte-rendu cette masse dans le tableau (mesure n°1) ;
    • Vider l’instrument et le remplir à nouveau d’eau (ne pas effectuer la tare une nouvelle fois, elle a déjà été réalisée avec l’instrument bien sec) ;
    • Noter sur votre compte-rendu la masse dans le tableau (mesure n°2) ;
    • Recommencer encore une fois la mesure.

    Estimation de l’ordre de grandeur de l’erreur sur la mesure#

    Pour mieux caractériser la dispersion des mesures, on calcule l’écart- type, noté \(\sigma_{n−1}\).

    L’écart-type \(\sigma_{n−1}\) donne une estimation de l’ordre de grandeur de l’erreur sur la mesure. Plus l’écart-type est faible, moins les valeurs mesurées sont dispersées, et donc plus précise est la mesure.

    Ces calculs peuvent être effectués avec la calculatrice, mais également avec Python :

    1. Importer la bibliothèque numpy qui permet d’effectuer des statistiques avec l’instruction :

      1
      import numpy as np

    2. Pour calculer l’écart-type des valeurs de la liste masse_becher et l’afficher :

      1
2
      ecart_typeB = np.std(masse_becher, ddof = 1)
print("Bécher : écart-type = ", round(ecart_typeB, 3))

      Compléter le script afin d’afficher les écart-types correspondant aux 3 instruments de mesure. L’instrution plt.show() doit se trouver en fin de script (instruction bloquante)

    3. Effectuer une capture d’écran de la console dans le compte-rendu (pour les valeurs des 3 écarts-type)

      On peut aussi calculer l’incertitude-type que l’on notera u (vient de uncertainty en anglais) : L’incertitude–type u(m) sur la masse est égale à :

      \[u(m ) = \dfrac{\sigma_{𝒏−𝟏}}{\sqrt{n}}\]

      Où \(n\) correspond au nombre de mesures effectuées. L’incertitude est toujours exprimée avec un seul chiffre significatif arrondi à la valeur supérieure. L'incertitude-type caractérise la dispersion des valeurs attribuées à la valeur mesurée (mesurande).

    4. Pour les 3 instruments utilisés, calculer l'incertitude type.

    Validation du choix de l'instrument le mieux adapté à la mesure du volume.#

    Les histogrammes et les valeurs des incertitudes-type doivent vous permettre d'effectuer les choix suivants :

    1. Pami les 3 instruments utilisés, lequel ne doit surtout pas être utilisé pour mesurer un volume ?

    2. Parmi les 2 instruments restants, lequel est le plus précis ? Quel est alors l'utilité de l'autre ?
